# bio-catalog

### Biological species catalog

<br>

#### Cloning repo
````bash
$ git clone git@github.com:jlucasps/bio-catalog.git
$ cd bio-catalog
````

----

### Run development

````bash
# build the app
$ npm install
$ bower install
$ grunt build

# run node server:
$ npm start

# If nodemon instaled
$ nodemon

````
